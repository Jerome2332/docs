TGPC Documentation

This documentation is built with Mintlify

Mintlify docs require their own hosting platform to function properly.

Setup Instructions

  1. Go to mintlify.com
  2. Sign up/login with your GitHub account
  3. Connect the Jerome2332/docs repository
  4. Your docs will be automatically deployed to a URL like:
    your-name.mintlify.app

Once deployed through Mintlify, all your documentation including the new SPINABT page will be accessible with full functionality.
